Virtual Drums はパントマイムに似た動きで演奏が可能なグローブ型のデバイスであり、
「第三回 100Project」にて製作を行なった。
当 Project では Virtual Drums チームリーダーの役職に就き、今作品の着想から最終プレゼンまでを務めた。
Arduino で MIDI 音楽ソフトを動かすスクリプトや赤外線レーザー測距での距離調整等、様々なプロトタイプを都度製作し、先輩から助言をいただきながら改良を続けた。
